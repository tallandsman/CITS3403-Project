How to play

Summary

Shark Atteck rules are very simple. The board is divided into cells, with sharks randomly distributed. To win, you need to open all the cells. The number on a cell shows the number of sharks adjacent to it. Using this information, you can determine cells that are safe, and cells that contain sharks. Cells suspected of being mines can be marked with a flag using the right mouse button.

Sharks

Scary sharks are in this area, you need to spot sharks to keep the beach safe!

Numbers

The numbers indicates the number of sharks you to find them

Flags

Using the flags will help you to keep track of where you think the sharks are

Winning

You've won the game when you've opened all squares that don't contain a shark. If you've opened all the empty/nr squares but haven't flagged the mine squares remaining, they will be auto-flagged and you have won. So, essentially flagging is not required, it's only there to help you keep track of where you think the mines are. The real way to win is open all the non-shark squares.
